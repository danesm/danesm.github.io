2.COMMAND key shortcuts

The command key on the Mac keyboard, with a ⌘ symbol, works very much like the control (Ctrl) key on the Windows keyboard. For example, the Mac keyboard shortcut to copy is "⌘-C" which is similar to the "Ctrl-C" shortcut you may be used to from Windows. Many of the command ⌘ and character shortcuts on the Mac are exactly the same as the Ctrl and character shortcuts on Windows, such as Copy, Cut, and Paste:

Key Combination Result
command ⌘+C Copy the selected data to the Clipboard
command ⌘+X Cut
command ⌘+V Paste the Clipboard contents
command ⌘+A Select all items or text in the current window
command ⌘+F Open a Find window or find text in a document
command ⌘+G Find the next occurrence of the selection
command ⌘+shift+G Find the previous version of the selection
command ⌘+N Create a new file in the current app
command ⌘+O Open an existing file in the current app
command ⌘+P Print the current file
command ⌘+Q Quit the current app
command ⌘+S Save the current app
command ⌘+shift+S Display the Save As dialog box
command ⌘+W Close the current window
command ⌘+option+W Close all windows in the current app
command ⌘+Z Undo the previous command
command ⌘+shift+Z Redo, puts back the last change made using undo
command ⌘+spacebar Show or hide the Spotlight search field
command ⌘+option+spacebar Show the Spotlight search results window
command ⌘+tab Move forward to the next most recently used app in a list of open apps
command ⌘+Mission Control (F3) Show the Spotlight search results window
command ⌘+minus (-) Decrease the size of the selected item
command ⌘+colon (:) Display the Spelling and Grammar window
command ⌘+semicolon (;) Find misspelled words in the document

3.Multi-Touch Gestures

The Mac has a trackpad, which supports multi-touch gestures. You can move your finger across the trackpad to guide the cursor, and the entire trackpad surface can be pressed like a button.

Gesture Action
One-finger swipe Moves the cursor on the screen
One-finger press, lower right corner Right-click (typically displays pop-up context menu)
Two-finger swipe Scroll (up, down, left, or right)
Two-finger press Right-click (typically displays pop-up context menu)
Three-finger swipe Switch spaces or open apps
Three-finger double tap Look up a word in the dictionary or on Wikipedia
Four-finger swipe up Show Mission Control
Four-finger swipe down Show all windows for the current app
Four-finger pinch Show Launchpad
Four-finger spread Show Desktop

Few More HINTS FOR WINDOWS USERS:

1. To right-click when using a single-button mouse, hold down Control and then click.
2. To right-click on a laptop or track pad, click with two fingers resting on the trackpad. Control-click also works.
3. The top row on Mac keyboards is used for special controls, such as audio volume and brightness. To use them as function keys, hold down the Fn key. You can reverse this behavior by choosing System Preferences@@→Keyboard. The Keyboard preferences pane also has a complete list of keyboard shortcuts.
4. Closing an application’s last window usually doesn’t close the application itself.
5. Apple applications and many third-party Mac applications save your edits automatically while you work and keep track of different versions.
6. To rename a file in a Finder window, click the filename once and then press Return. The filename becomes editable.
