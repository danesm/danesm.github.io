1. IBM Websphere App server (WAS) Introspection using JACL scripts. (✅ Click here to see the post)

This is something that I wrote about 9 years ago while learning WAS wsadmin scripting using JACL scripting. Jacl, Java Command Language, is a version of the Tcl scripting language for the Java environment. I used this tool to introspect WAS applicaton configuration on route to live servers for pre and post deployment checks. The WAS wsadmin tool supports two scripting languages: Jacl and Jython. Five objects are available when you use scripts: AdminControl: Use to run operational commands. AdminConfig: Use to run configurational commands to create or modify WebSphere Application Server configurational elements. AdminApp: Use to administer applications. AdminTask: Use to run administrative commands. Help: Use to obtain general help. To learn more about WAS wsadmin scripting please click here.
